# Kara3 Pricing & Profit Calculator

Internal pricing tool for Kara3 jewelry business. Calculate pricing, profit margins, break-even metrics, and ad spend viability for jewelry products.

## Features

- **Core pricing analysis** — final price, total costs, profit before/after ads, margins
- **Break-even metrics** — break-even CPA, break-even ROAS, minimum viable price
- **Decision support** — business verdict (Strong / Acceptable / Weak / Loss-Making), discount room, warnings
- **Scenario comparison** — no ads, moderate ads, aggressive ads side-by-side
- **Preset products** — Moissanite Ring, Zultanite Necklace, Silver Bracelet, Gold-Plated Earrings
- **Multi-currency** — EUR, USD, TRY, GBP
- **VAT toggle** — include or exclude VAT in calculations
- **Export** — copy summary to clipboard or download as TXT
- **Live calculation** — results update as you type
- **Mobile-friendly** — responsive dark premium design

## How to Use

1. Open `index.html` in a browser (or deploy to GitHub Pages)
2. Enter your product costs, fees, and pricing details
3. Results update live — or click **Calculate**
4. Use **preset buttons** to load sample jewelry products
5. Review the verdict, warnings, and scenario cards
6. **Copy Summary** or **Download as TXT** to share results

## GitHub Pages

This is a static HTML/CSS/JS app. To deploy on GitHub Pages:

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to the branch and folder containing these files
4. The calculator will be available at `https://<username>.github.io/<repo>/kara3-calculator/`
